# OpenCORE 2.0

Bienvenido a la versión 2.0 de OpenCORE. Esta es una reescritura moderna utilizando tecnologías web estándares y optimizada para **Vite**.

## Requisitos
Para ejecutar este proyecto correctamente (con módulos ES6 y rutas absolutas), necesitas **Node.js** instalado.
- [Descargar Node.js](https://nodejs.org/)

## Instalación y Ejecución
1.  Abre una terminal en esta carpeta.
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Inicia el servidor de desarrollo:
    ```bash
    npm run dev
    ```
4.  Abre la URL que aparece en la terminal (usualmente `http://localhost:5173`).

## Estructura del Proyecto
-   **index.html**: Página principal.
-   **src/css/**: Estilos modernos con variables CSS y Glassmorphism.
-   **src/js/**: Lógica de interacción (Scroll, Parallax, Formulario).
-   **src/assets/**: Imágenes migradas (Opencore Web/Mobile).

## Notas sobre GitHub
Debido a que `git` no está instalado en este entorno, no se pudo realizar la subida automática. Para subirlo:
1.  Instala Git.
2.  Ejecuta:
    ```bash
    git init
    git add .
    git commit -m "Init Opencore 2.0"
    git remote add origin <TU_URL_DEL_REPO>
    git push -u origin main
    ```
